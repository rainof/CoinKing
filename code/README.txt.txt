How to use
1. Launch the program "money_detector.m"
2. Click "Run"
3. If the path of image error 
	3.1 Click "Browse for Folder"
	3.2 Select folder "TotalCoin_Project"
	3.3 Click "Run" again
	3.4 When "MATLAB Editor" show, Click "Add to Path"
	3.5 Click "Run" again, The program can run :)
